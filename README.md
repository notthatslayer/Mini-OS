Mini OS

A browser-based desktop environment built using HTML, CSS, AND JavaScript.

This project recreates the feel of a simple operating system inside the browser. Users can open applications, manage windows, take notes, draw on a canvas, perform calculations, use terminal commands, and interact with a desktop-style interface.
